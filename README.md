# CDKDemo AWS CDK Infrastructure

This project provides an extendable template for deploying OxRSE projects to AWS.

It includes a NextJS application with a database and a user-defined 3rd party API key.

The deployment code is in [`aws_cdk`](./aws_cdk).

The app is in [`app`](./app) and allows users to caption photos of cats and dogs retrieved from 3rd party APIs.
